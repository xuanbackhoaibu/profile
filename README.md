# Portfolio Trần Xuân Bắc

Portfolio cá nhân dùng để giới thiệu hồ sơ, CV, kỹ năng và các dự án nổi bật với nhà tuyển dụng.

## Link

- Website: https://xuanbackhoaibu.github.io/profile/
- CV online: https://xuanbackhoaibu.github.io/profile/files/CV-Tran-Xuan-Bac.html
- CV PDF: https://xuanbackhoaibu.github.io/profile/files/CV-Tran-Xuan-Bac.pdf

## Công nghệ

- HTML5, CSS3, JavaScript
- CSS animation, scroll reveal, 3D card interaction
- Three.js cho hiệu ứng 3D ở hero
- GitHub Pages để public website

## Nội dung chính

- Giới thiệu và mục tiêu nghề nghiệp
- Mục "Tôi đang tìm kiếm" cho cơ hội thực tập Backend .NET, System Analyst, API Testing
- Kỹ năng backend, database, system analysis, API testing
- Dự án nổi bật và dự án bài tập học phần
- Dự án WebBanHangOnline đã có Docker Compose `webthoitrang`, SQL Server và dữ liệu demo
- CV online, CV PDF, thông tin liên hệ
- Favicon/avatar và Open Graph để khi chia sẻ link hiển thị đẹp hơn trên Facebook, Zalo, LinkedIn

## Chạy local

```bash
npm install
npm run build
```

Sau khi build, mở file `index.html` trực tiếp trong trình duyệt hoặc dùng một static server bất kỳ.

## Ảnh demo

Ảnh đại diện và hình nền đang nằm trong thư mục `assets/`.
